5.3.10. EID Constants

5.3.10. EID Constants

An EID constant is a value that represents an entry ID. An EID constant can be written as any series of five characters from the EID character map, surrounded by square brackets. These are used to refer to entries (or files) in levels.

Here are some examples of valid EID constants: [WillG] [sp23A] [BoxsC] [do13A]
