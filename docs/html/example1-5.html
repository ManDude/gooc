3.1.5. Compiling

3.1.5. Compiling

Here's how to compile the GOOC source code file into a GOOL executable file:

gooc -c 2 DbgOC.gooc DbgOC.nsentry

.nsentry is the conventional extension used for exported entries in CrashEdit.

Just place the resulting entry in the game, patch any other files you may need to patch, decide how you want to spawn (such as with an entity) and you're good to go.

If you would like to spawn this object from others stored in different GOOL executables, you will need to use the following command instead:

gooc -c 2 -h DbgOC.gooh DbgOC.gooc DbgOC.nsentry

This creates a header file which contains some expression macros for the spawns and GOOL ID, so that their names can be used instead of having to type in numbers manually and hope they're correct.
