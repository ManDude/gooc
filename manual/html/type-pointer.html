6.3. Pointers

6.3. Pointers

Pointers are integers that hold memory addresses to a certain structure or value. These are commonly used to store the address of an object in memory, in fact, the first eight GOOL fields (link fields or link variables) are specifically for storing pointers to objects. Pointers are functionally identical to integers, in that they can be used in arithmetic, passed as parameters, and so on. The main difference comes from how you create one. You can create a pointer by using the address-of & operator on any valid REPLACEME gool reference. Arrays are already pointers, so the address-of operator is not necessary for them. Outside of referencing other objects, pointers are rarely used, only being useful for specific scenarios such as certain instructions or variadic arguments. You cannot create a pointer to a global variable.
