NAME

gooc — Crash Bandicoot game-oriented object C script compiler tool

SYNOPSIS

gooc [−V] [−c version] [−r region] [−h output] [−e format] ... [input [output]]

DESCRIPTION

The gooc utility compiles gooc scripts. These options are accepted:

gooc −c version [
−r region] [
−h output]... [input [output]]

Compiles an enemy script.

−V

Displays the program version and exits.

−c version

Compiles a script.

−r region

Sets the region mode, used for time based units. "ntsc-u", "ntsc-j" and "pal" are acceptable. Defaults to "ntsc-u".

−h output

Set the output path for a "header" file, which can be included in gooc scripts, and contains the resulting values of the GOOL object ID and its spawns.

−e format

Set the output path format for external GOOL modules. For example. using format ‘%s.gool’ and a module named ‘TestC’ will result in an output directory of ‘TestC.gool’ for this particular module.

Replace the version option by the gool script format version requested. Running the program without a command will list the supported formats.

EXIT STATUS

The gooc utility exits with 0 on success, 1 on error.

SEE ALSO

Project homepage: https://github.com/ManDude/thtk/tree/gooc

SECURITY CONSIDERATIONS

Invalid data may not be properly handled. Do not operate on untrusted files.
